# 🤖 Minecraft AFK Monitor Bot

Бот для Minecraft, который стоит в AFK, мониторит вход нужного игрока, делает скриншоты и отправляет уведомления в Telegram с видео!

## ✨ Возможности

- 🎯 Отслеживание входа игрока с нужным ником
- 📸 Автоматические скриншоты каждые N секунд
- 🎬 Создание видео из скриншотов по команде
- 📱 Уведомления в Telegram с точным временем
- 🔄 Автоматическое переподключение при обрыве
- 📝 Логирование всех событий
